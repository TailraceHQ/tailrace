---
title: tRPC integration
description: Govern tRPC procedure input and output at the tool boundary.
mode: guide
---

```ts
import { createTailrace } from "@tailrace/core";
import { createTailraceMiddleware } from "@tailrace/trpc";
import { initTRPC } from "@trpc/server";

const t = initTRPC.create();
export const procedure = t.procedure.use(
  createTailraceMiddleware(createTailrace(), { agent: "api" }),
);
```

Non-streaming queries/mutations only in v0.1. Full write-up: `docs/guides/trpc-integration.md`.
