---
title: Fastify integration
description: Add Tailrace OpenAI-compatible plugin to Fastify.
mode: guide
---

```ts
import { createTailrace } from "@tailrace/core";
import { tailraceFastify } from "@tailrace/fastify";
import Fastify from "fastify";

const app = Fastify();
await app.register(tailraceFastify(createTailrace()));
```

Full write-up in the repo: `docs/guides/fastify-integration.md`. Reference: [/docs/reference/fastify](/docs/reference/fastify).
