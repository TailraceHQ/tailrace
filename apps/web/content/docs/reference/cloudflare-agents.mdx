`@tailrace/cloudflare-agents` Compose helpers for Cloudflare Agents. Model/tool wrapping delegates to `@tailrace/ai-sdk`.

## Install

```bash
pnpm add @tailrace/core @tailrace/cloudflare-agents ai
```

Peers: `ai@^5` (required); `agents` optional.

## Exports

| Export                      | Description                          |
| --------------------------- | ------------------------------------ |
| `createCloudflareTailrace`  | Tailrace with kvVault or memoryVault |
| `withCloudflareAgents`      | Fluent `forChat` + `wrapOnToolCall`  |
| `CloudflareTailraceOptions` | Identity / vault options             |
