`@tailrace/fastify` provides an OpenAI-compatible Fastify plugin (`fastify` `>=4`).

```ts
await app.register(tailraceFastify(tailrace));
```

Same 422 / SSE contract as Hono. Guide: [Fastify integration](/docs/guides/fastify-integration).
